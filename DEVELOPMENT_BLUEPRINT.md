# Development Blueprint for 2D BGM GameDev

## Project Structure
- **src/**: Contains all source files.
  - **components/**: Reusable components used throughout the project.
  - **assets/**: Game assets including images, sounds, and music.
  - **scenes/**: Different game scenes (like the main menu, levels, etc.).
- **tests/**: Unit and integration tests for various components.
- **docs/**: Documentation related to the project.

## Completed Features
1. **Character Movement**: Players can move characters using keyboard inputs.
2. **Basic Scoring System**: Points are awarded based on player actions.
3. **Level Transitions**: Smooth transitions between different game levels.
4. **Audio Integration**: Background music and sound effects are integrated and functioning.

## Implementation Details
- **Environment**: Developed using Unity with C# as the main programming language.
- **Version Control**: All code is maintained using Git, with branches for features and fixes.
- **Testing Framework**: Implemented using NUnit for unit tests, ensuring all game features are functioning as intended.
